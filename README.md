## Objective
To understand basic cloud infrastructure by launching and managing an AWS EC2 instance using Free Tier.

## Tools Used
- AWS EC2 (Free Tier)
- Amazon Linux 2
- Apache HTTP Server
- SSH

## Steps Performed
1. Created AWS Free Tier account
2. Launched Amazon Linux EC2 instance (t2.micro)
3. Configured key pair and security group
4. Connected to EC2 using SSH
5. Installed and started Apache web server
6. Accessed application using public IP
7. Safely stopped the EC2 instance

## Deliverables
- EC2 instance running screenshot
- SSH connection proof

## Outcome
Gained hands-on understanding of cloud basics, EC2, security groups, key pairs, and public/private IPs.
